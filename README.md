# NGS Trefftz
Trefftz and quasi-Trefftz functions for the acoustic wave equations implemented for NGSolve

[![arXiv](https://img.shields.io/badge/arXiv-1907.02367-b31b1b.svg)](https://arxiv.org/abs/1907.02367)
[![arXiv](https://img.shields.io/badge/arXiv-2011.04617-b31b1b.svg)](https://arxiv.org/abs/2011.04617)

depends on [NGSolve](https://github.com/NGSolve/ngsolve) 

install using 
```
mkdir make && cd make
cmake ..
make install 
 ```
